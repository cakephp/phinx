Phinx 文档
===================

Phinx 使得管理 PHP 程序的数据库迁移变得非常容易。您可以在仅仅5分钟内，通过 Composer 安装 Phinx 并创建您的第一个数据库迁移。Phinx 只会专注迁移，不会臃肿的涉及数据库 ORM 系统或应用程序框架。
